2012-10-17

  • version 0.1.8
  • added some opitons
    • yaft reads environment value, YAFT
      • wall: display wallpaper
      • w=N: set terminal width (pixel)
      • h=N: set terminal height (pixel)
      • x=N: set x offset
      • y=N: set y offset
$ export YAFT="wall w=640 h=384 x=32 y=32"; yaft
