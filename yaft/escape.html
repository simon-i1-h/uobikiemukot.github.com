supported escape sequence by yaft

Control Characters

hex char function
0x08 BS backspaces one column
0x09 HT goes to the next tab stop or to the end of the line if there is no earlier tab stop
0x0A LF gives a linefeed
0x0B VT same as LF
0x0C FF same as LF
0x0D CR gives a carriage return
0x1B ESC starts an escape sequence

ESC

sequence function
ESC 7 saves current state (cursor coordinates, attributes and DECOM)
ESC 8 restores state most recently saved by ESC 7
ESC D gives a linefeed
ESC E gives a newline (CR + LF)
ESC H sets a tab stop at current column
ESC M gives a reverse linefeed
ESC Z DEC private identification (answer: ESC [ ? 6 c I am a VT102)
ESC [ control sequence introducer (CSI)
ESC ] operating system command (OSC)
ESC c resets a terminal state (mode, scrolling region, cursor coordinates, current color pair, attributes and tabstop)
