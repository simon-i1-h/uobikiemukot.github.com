glyph width report (GWREPT)

request glyph width list

OSC 8900 ; Ps ; ? ST
  • Ps: reserved (maybe charset)

answer

OSC 8900 ; Ps ; Pt ; width : from : to ; width : from : to ; ... ST
  • Ps: reserved (maybe charset)
  • Pt: reverved (maybe East Asian Width version)
  • width: 0 or 1 or 2
  • from, to: unicode code point (decimal)
